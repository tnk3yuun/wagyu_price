# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- リモートリポジトリは複数人で共有するためのリポジトリ
- ローカルリポジトリは個々で使用するために自分のPC上に配置するためのリポジトリ

## プッシュとマージの違いは何でしょうか？

- プッシュはローカルリポジトリで作成したリモートリポジトリにアップロードさせることでマージはいくつかの違う履歴を1つに融合させること


## コミットとプッシュの違い
- コミットは自分のPCに保存されるがプッシュは大きい範囲で保存される


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
- git commit -m '変更したい名前'


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
- ローカルでマージする場合は自身で作業ブランチに取り込むことができるが、プルリクエストは自身で行うことができない


## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- ソースコードを書いた人と相談する。
1. 先にマージされた変更内容を取り込む
2. 後にマージしようとしている変更内容を取り込む
3. どちらの変更内容も取り込む
